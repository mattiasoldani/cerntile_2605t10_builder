# cerntile_2605t10_builder
Event-building software for the ALLEGRO TileCal first beamtest data (from the May 2026 data taking on the CERN T10 beamline). QC_scripts and their dependencies are courtesy of Jiri &mdash; [GitLab](https://gitlab.cern.ch/jkvas), [GitHub](https://github.com/jkvas). In particular, check out his [DT5202_Raw_Utils](https://gitlab.cern.ch/jkvas/dt5202_raw_utils).
